# 🛡️ SafeStreet AI

> AI-Powered Personal Safety & Emergency Response System

## 📌 Overview

SafeStreet AI is an intelligent personal safety application designed to proactively detect potential emergencies and assist users during critical situations. Unlike traditional safety apps that rely only on a manual SOS button, SafeStreet AI combines smartphone sensors, contextual information, and AI-based risk analysis to identify dangerous situations and automatically notify trusted contacts.

The application is designed for:
- 👩 Women
- 👴 Elderly People
- 🧒 Students
- 🚶 Solo Travelers
- 🚚 Delivery Personnel
- 🌙 Night Shift Employees

---

## 🚀 Problem Statement

In many emergency situations, victims are unable to manually open an application and press an SOS button. Existing safety applications depend heavily on user interaction.

SafeStreet AI aims to solve this problem by intelligently detecting abnormal situations and initiating emergency assistance automatically while reducing false alarms.

---

# ✨ Key Features

## 🧠 1. Guardian AI (Smart Danger Detection)

Analyzes multiple factors such as:
- Motion patterns
- Sudden phone movement
- User inactivity
- Journey status
- Time and location

Generates an AI Risk Score before triggering an emergency.

---

## 🚨 2. Instant SOS

Automatically or manually sends:
- Live GPS location
- Emergency alert
- User details
- Time of emergency

to trusted contacts.

---

## 🛣️ 3. Journey Shield

Allows users to start a monitored journey.

Features:
- Live trip monitoring
- Safety check reminders
- Detects unexpected stops
- Route tracking

---

## 📍 4. SafeFinder

Displays nearby:
- Police Stations
- Hospitals
- Pharmacies
- Emergency Services

using Google Maps.

---

## 📶 5. Offline Rescue

If internet is unavailable:
- Continues monitoring locally
- Uses SMS when possible
- Stores emergency data
- Automatically syncs when connectivity returns
